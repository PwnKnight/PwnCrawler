# PwnCrawler - Web Crawler
## Description

PwnCrawler is a lightweight Python web crawler that allows users to discover all internal URLs on a website. It helps in security research, website mapping, and automated data collection for ethical hacking purposes.

## Features

Crawl a website to discover all internal URLs

Save the crawled URLs in a JSON file named DOMAIN_pwncrawler.json

## Prerequisites

- Python 3.6 or above
- Pip (Python package manager)

## Installation

Clone the repository:

- git clone https://github.com/PwnKnight/PwnCrawler

- cd PwnCrawler

## Install dependencies:

pip install -r requirements.txt

## Usage
python3 pwncrawler.py <URL>

## Example:

python3 pwncrawler.py https://example.com


This will create a file named example.com_pwncrawler.json containing all discovered URLs.
